### UI UX Pro Max Skill

| Field | Detail |
|---|---|
| **Category** | UI / UX Design Intelligence |
| **Use When** | Building any website, landing page, dashboard, or user interface |
| **Original** | https://github.com/nextlevelbuilder/ui-ux-pro-max-skill |
| **Lynn's Fork** | https://github.com/APXprojects/ui-ux-pro-max-skill |

**What it does:**
Analyzes the project type and auto-generates a complete design system before any code is written.
Covers visual style selection, color palettes, typography pairings, layout patterns, and anti-patterns to avoid.

**What to apply from it:**
- Match the project type to one of its 100 industry reasoning rules (SaaS, consulting, wellness, fintech, etc.)
- Select an appropriate style from its 67 UI styles
- Apply a matching color palette from its 96 options
- Use a curated font pairing from its 57 typography combinations
- Run its pre-delivery checklist before finalizing any UI

**Relevant ventures:**
CASA, CORTEN Advisory, BPM Services, VERRADO, Only One Name, Brooke's agency, INEVITABLE
